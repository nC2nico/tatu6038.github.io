フォントのインストール方法！

今回は初の記事作成となるので下手なところやミスがあっても温かい目で見守ってください。
というわけで今回はwindows10,mac,androidでのフォントの導入方法についてご紹介していきたいと思います。

Windowsでの導入方法

  1. 検索等でダウンロードしたフォントの入ったファイルを解凍します。
  2. エクスプローラーを起動し[C:]>[Windows]>[fonts]の順に開きます。
  3. ステップ１で解凍したフォルダーの中の.ttfや.otfなどの拡張子のフォントのファイルを全て選択し、ステップ２で開いたフォルダーにドラッグアンドドロップします
  4. 完成！
    (注: ステップ３前に開いていたアプリでの使用には本体の再起動かアプリの再起動が必要です)

Mac

  1. 検索等でダウンロードしたフォントの入ったファイルを解凍します。
  2. 解凍したフォルダ内の.ttfや.otfなどの拡張子のフォントファイルをfontbookで開きます(普通にダブルクリックでも可)。
  3. 起動したらウィンドウ(?)の下の[インストール]をクリック
  4. 完了！
    再起動必要なのかはわからん

Android

  1. Androzip(これでやると楽かと)というアプリをダウンロードします。
  2. Phonto(同様)というアプリをダウンロードします。
  3. Androzipで検索等でダウンロードしたフォントの入ったファイルを解凍します。
  4. [アクションを選択]から[開いてください]を選択。
  5. [アプリケーションを選択]から[Phonto]を選択
  6. [インストール]を押下
  7. 完了！
    (こちらも同様、再起動必要なのかはわからん)
